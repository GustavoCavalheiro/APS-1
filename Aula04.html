Parágrafos e elementos de marcação (alguns)

Elemento p:

O elemento p é utilizado para marcar o início e o /p é utilizado para marcar fim de um parágrafo.

Elemento br:

O elemento br é usado para quebra de linha.

Elemento hr:

O elemento hr é usado para criar uma linha horizontal.

Elemento u:

O elemento u é usado para sublinhar um texto.

Elemento i:

O elemento i é usado para deixar um texto em itálico.

Elemento b:

O elemento b é usado para deixar um texto em negrito.

Elemento strong:

O elemento strong é usado para deixar um texto em negrito.

Elemento em:

O elemento em é usado para deixar um texto em itálico.

Elemento mark:

O elemento mark é usado para marcar um texto.

Elemento small:

O elemento small é usado para diminuir o tamanho do texto.

Elemento pre:

O elemento pre é usado para manter o texto com a formatação original.
